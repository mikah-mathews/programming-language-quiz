You should learn Javascript!


Javascript is a varied and useful language, great for adding functions to websites and making them more interactive.

It is an excellent language to start out with due to how many resources there are! It is widely used in almost every website and you can code right in your browser!

If you want to make a website, get user input, make a browser game, create better user interactions, or work on hunting down bugs in mass databases, there's always a demand for Javascript!

A good resource to learn Javascript is at Free Code Camp

You should learn Python!


Python is an amazing, flexible language great for all kinds of projects.

It is often used in back end development for servers, but you can also use it in web development, security, and machine learning!

You are able to create functioning pieces of code with less work and be able to focus on creating the end project.

Python is an excellent base for starting to program and is fun to learn!

A good resource to learn Python is at this free Udemy course.

You should learn Ruby!


Ruby is a popular language, designed for humans instead of the other way around.

It is commonly paired with Rails, often both refered to as 'Ruby on Rails'. Ruby on Rails allows you to quickly build web applications through pre-made framework.

Startups love using this language due to it's ease of implementing new features at a rapid pace.

Ruby and Ruby on Rails both have large communities which makes learning and asking questions even easier!

A good place to learn Ruby is at Ruby Monk
